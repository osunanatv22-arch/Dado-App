# Dado App (Streamlit)

App web en Streamlit para tirar un dado de 6 caras N veces y visualizar:
- Histograma de resultados
- Gráfico de dispersión (tirada vs valor)
- Controles para cantidad de tiradas y modo de re-lanzar

## Ejecutar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
